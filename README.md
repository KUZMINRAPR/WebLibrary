# Проект, в котором делается система библиотеки.

--- 

Разделен на проекты:
 - WebLibrary.Front (Blazor WASM) - Фронтенд библиотеки (Пользовательский интерфейс)
 - WebLibrary.API (Web API) - Здесь лежат: 
   - контроллеры, 
   - подключения DI,
   - Swagger
 - WebLibrary.Domain (Ядро) - Здесь лежат: 
   - Entities (сущности с идентификаторами), 
   - Interfaces (интерфейсы), 
   - Exceptions (исключения уровня домена), 
   - ValueObjects (сущности без идентификаторов)
 - WebLibrary.Application - Здесь лежат: 
   - Commands,
   - Queries,
   - Handlers (для команд и запросов),
   - DTO,
   - Validators (FluentValidation),
   - Pipeline Behaviours (валидация, логирование) 
 - WebLibrary.Infrastructure - Здесь лежат: 
   - EF Core + миграции,
   - реализация репозиториев,
   - конфигурация ORM

Используемые технологии:
 - ASP.NET Core 9.0
 - Entity Framework Core 9.0
 - Blazor 9.0
 - Sqlite
 - Swagger

Реализовано:
 - Поиск книги
 - Добавление книги
 - Получение всех книг из базы данных библиотеки
